# Writerly Agent

An interactive writing development studio built with Next.js. Choose a craft focus, draft in-app, and receive live agent guidance on clarity, rhythm, tone, and deliberate practice drills.

## Features

- Dynamic prompt generator across four focus tracks (Narrative Clarity, Persuasive Impact, Creative Voice, Concise Synthesis)
- Real-time draft analysis covering readability, pacing, vocabulary freshness, tone balance, filler detection, and sentence-level flags
- Session blueprint with warm-up, sprint, and revision phases plus micro-drills to reinforce deliberate practice
- Reflection prompts and growth levers to anchor weekly progress
- Tailored to deploy seamlessly on Vercel

## Getting Started

First, run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
